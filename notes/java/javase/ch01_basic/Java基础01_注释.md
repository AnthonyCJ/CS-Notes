# Java基础_注释

---

==**书写注释是一种良好习惯**==

* Java 的注释包括三种
  * 单行注释
  * 多行注释
  * 文档注释

## IDEA 中设置注释颜色风格

位置：Settings -> Editor -> ColorScheme -> Java 其中包含了Java各类变量的颜色风格设置。









## 新建项目注意事项

* 新建一个空项目时，要为项目新建一个模块（new Module）。步骤如下

  1. File -> new -> Modules，新建一个模块（如：基础语法）

  2. File -> Project Structure -> Project，设置项目 SDK 和对应的 language level。
